API documentatie

Dit is de documentatiepagina voor mijn Node.js + Express API. De API omvat twee entiteiten: users en posts.

Users endpoints

  • GET /api/users - Alle users ophalen
  • GET /api/users/:id - Eén specifieke user ophalen op basis id
  • POST /api/users - Nieuwe user toevoegen
  • PUT /api/users/:id - Bestaande user updaten
  • DELETE /api/users/:id - Bestaande user verwijderen
  • GET /api/users/sort?sort=&order= - Users sorteren op kolom en volgorde

Posts endpoints

  • GET /api/posts - Alle posts ophalen
  • GET /api/posts/:id - Eén specifieke post ophalen op basis van id
  • POST /api/posts - Nieuwe post toevoegen
  • PUT /api/posts/:id - Bestaande post updaten
  • DELETE /api/posts/:id - Bestaande post verwijderen
  • GET /api/posts/paged?limit=&offset= - Posts ophalen met paginatie
  • GET /api/posts/search?term= - Posts zoeken op titel of inhoud

Parameters

  • :id - Dynamische parameter voor specifieke user of post
  • limit - Aantal resultaten dat wordt teruggeven
  • offset - Startpositie in de lijst met resultaten
  • term - Zoekterm voor de search endpoints
  • sort - Kolomnaam om op te sorteren (bv. name, email, created_at)
  • order - Sorteervolgorde: asc of desc

Voorbeelden

  • GET /api/posts/paged?limit=5&offset=0
  • GET /api/posts/search?term=Wesley
  • POST /api/posts met JSON body: {"title": "...", "content": "...", "user_id": 1}
  • GET /api/users/sort?sort=first_name&order=asc
